## Installation
> Create a Conda Python environment and install requirements.
```
conda create -n vione python==3.10
conda activate vione
cd vione
pip install -r requirements.txt
```

## Specifications
- Capacity: 34.3GB
- RAM: 24GB more
- GPU: (Tested: RTX4080 and A100)
